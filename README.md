Current Features:
* Search database of recipes
* See a surprise recipe (random)
* Weeknight Favorites
* Recent Recipes
* Seasonal Recipes
* Report an issue with a recipe
* Favoriting/Bookmarking using local storage
* Send recipe to a friend
